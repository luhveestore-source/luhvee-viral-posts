## 📊 Interpretando Google Trends

### O Que Significam os Números?

```
Interesse: 0-100 (relatividade)
├─ 100 = Pico de interesse
├─ 50 = Metade do pico
└─ 0 = Sem volume de busca

Tendência:
├─ ⬆️ +50% = Crescendo MUITO
├─ ⬆️ +20% = Crescendo
├─ ⬇️ -10% = Caindo
└─ ⬇️ -50% = Caindo MUITO
```

### Quando Usar Cada Nível

```
⬆️ +70%+ = FOMO + Urgência 🚀
         (Todos procurando!)

⬆️ +30-70% = Urgência + Curiosidade 📈
           (Trending forte)

⬆️ +10-30% = Desconto + Social Proof 💰
           (Bom interesse)

⬇️ Negativo = Exclusividade + Storytelling 👑
           (Público selecionado)
```

## 💡 Tips & Tricks

### ⚡ Aproveitar ao Máximo

**1. Trending Recente**
```
Produto acabou de aparecer:
├─ Use Curiosidade (desperta)
├─ Depois use Urgência (cria pressa)
└─ Depois use FOMO (medo de perder)
```

**2. Trending Consolidado**
```
Produto já é popular:
├─ Use Social Proof (recomendação)
├─ Desconto (valor)
└─ Exclusividade (selecionado)
```

**3. Trending em Queda**
```
Produto caindo em buscas:
├─ Use Exclusividade (raro)
├─ Curiosidade (único)
└─ Ofereça desconto (último chance)
```

**Sex:** ⬆️ +10% → Curiosidade

## ⚠️ Cuidados & Limitações

### ✅ Funciona Bem
```
✅ Produtos em alta demanda
✅ Nichos específicos (moda, beleza)
✅ Tendências sazonais (Natal, verão)
✅ Produtos novos em alta
```

### ⚠️ Cuidado Com
```
⚠️ Muito genérico ("produto", "loja")
⚠️ Misspellings ("bolsa de ombr" vs "bolsa de ombro")
⚠️ Produtos muito obscuros (sem volume)
⚠️ Regional (trending muda por região)
```

### 🚫 Não Funciona
```
❌ Palavras vazias ("a", "o", "de")
❌ Buscas muito gerais
❌ Termos não indexados (nomes próprios)
❌ Volumes muito baixos
```

## 🐛 Troubleshooting

### "pytrends não instalado"
```bash
pip install pytrends
# Depois reexecute
streamlit run luhvee_vendas_pro_trending.py
```

### "Erro ao conectar Google Trends"
```
1. Verifica conexão internet
2. Google às vezes throttles
3. Aguarde 1 minuto
4. Tente novamente
5. Usa cache se falhar
```

### "Dados em cache são antigos"
```
1. Delete: trending_cache.json
2. Reexecute app
3. Clique "Buscar Tendências" novamente
4. Terá dados frescos!
```

### "Interesse zerado para um produto"
```
Significado:
├─ Não foi buscado essa semana
├─ Volume muito baixo
└─ Considere outro produto

Solução:
├─ Tenta com nome mais popular
├─ Ou category mais genérica
└─ Google Trends é 1 das 5 opções
```
