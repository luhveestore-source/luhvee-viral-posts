# 🚀 Quick Start - Google Trends Version

Setup em **5 minutos**! ⚡

---

## 1️⃣ Instalar Dependência

```bash
pip install pytrends
```

Só isso! Sem API Keys, sem configuração complicada.

---

## 2️⃣ Executar

```bash
streamlit run luhvee_vendas_pro_trending.py
```

Abre em: `http://localhost:8501`

---

## 3️⃣ Usar a Nova Aba

### Aba "Google Trends" (Nova!)

```
1. Clique: "🔄 Buscar Tendências"
2. Aguarde: ~30 segundos
3. Veja: Produtos viralizando AGORA
4. Clique: "Usar" no que quiser
5. Automático: carrega em "Gerar Copies"
```

**É isso!** 🎉

### 1. Timing é Tudo
```
Quando um produto entra trending:

- Você quer ser o PRIMEIRO

Google Trends te diz quando! ⏰

### 2. Combinar com Estratégia
```
Trending +70%? Usa FOMO
"Todos procurando por isso!"

"Raro, última chance!"

```

Próximas 1 hora: Instantâneo (cache)

Sistema otimizado! ✅
```

## ⚡ Troubleshooting

### "Importação pytrends falhou"
```bash
pip install --upgrade pytrends
# Reexecuta app
```

### "Google Trends não responde"
```
Google às vezes throttles
Espere 1 minuto
Tente de novo
Sempre tem cache local! ✅
```

### "Dados do cache estão antigos"
```
Delete: trending_cache.json
Reexecuta app
Busca novamente
Fresh data! 🆕
```
