Pegue este arquivo:
```

```

Cole na mesma pasta do seu projeto:
```

```

---

## PASSO 2️⃣ - Abrir Terminal

Abra o terminal/cmd na pasta do seu projeto

```bash
# Windows: cmd.exe
# Mac/Linux: Terminal
# Streamlit Cloud: não precisa
```

---

## PASSO 3️⃣ - Instalar Dependência

Copie e cole no terminal:

```bash
pip install --upgrade pytrends
```

**Aguarde terminar** ⏳

---

## PASSO 4️⃣ - Limpar Cache (Opcional mas Recomendado)

```bash
rm trending_cache.json
```

**Ou delete o arquivo manualmente se tiver**

---

## PASSO 5️⃣ - Executar App

```bash
streamlit run luhvee_vendas_pro_trending.py
```

**Aguarde abrir no navegador** 🌐

---

## PASSO 6️⃣ - Testar Google Trends

Na app que abriu:

1. Vá para aba: **"🌐 Google Trends"**
2. Clique botão: **"🔄 Buscar Tendências"**
3. Aguarde ~60 segundos (primeira vez)
4. Veja produtos viralizando! ✅
